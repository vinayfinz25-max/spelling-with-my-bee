# Milestone 1: Project Foundation

## Scope

- React 19 + Vite application shell.
- Strict TypeScript configuration.
- TailwindCSS theme with the approved palette.
- React Router route map.
- TanStack Query provider.
- Zustand preferences store.
- Framer Motion available for UI feedback.
- Supabase client dependency installed for later milestones.
- ESLint, Prettier, and Vitest tooling.

## Verification

Run these commands from the project directory:

```bash
pnpm lint
pnpm test
pnpm build
```

Milestone 2 should not start until all three pass.
