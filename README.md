# Spelling with my bee

A private browser word game for friends. Players open a URL, enter a nickname,
and play a daily letter puzzle or join a private troop room.

## Stack

- React 19
- TypeScript
- Vite
- TailwindCSS
- Framer Motion
- Zustand
- React Router
- TanStack Query
- Supabase
- Vitest

## Local Development

```bash
pnpm install
pnpm dev
```

## Verification

```bash
pnpm lint
pnpm test
pnpm build
```
